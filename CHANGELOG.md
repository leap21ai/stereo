# Changelog

## 0.1.0 (2026-03-12)

Initial release.

- Custom editor provider for `.md` files with executable code blocks
- Markdown parser: extracts `` ```lang run `` blocks with options
- VM sandbox execution with React SSR
- Built-in components: MetricCard, DashGrid, Sparkline, StatusTable
- Auto-refresh support via `refresh=` option
- OKLCH-themed dark UI matching VS Code
- Environment variable reference extraction (`{{env.X}}`)
